SANJEEV BATCHALI - STATIC WEBSITE EXPORT
=========================================

This folder contains a fully static HTML/CSS/JavaScript version of your portfolio website.

STRUCTURE:
----------
- index.html          Main HTML file
- styles/main.css     All styling (includes light/dark mode)
- scripts/main.js     All JavaScript functionality
- assets/             Folder for images (add your profile.jpg here)

FEATURES:
---------
✓ Cosmic wave animated background with twinkling stars
✓ Dark mode toggle (saves preference to localStorage)
✓ Rotating hero taglines
✓ Finance skills infinite ticker (8 seconds)
✓ Tech skills infinite ticker (20 seconds)
✓ Professional summary & industry exposure
✓ Work experience cards
✓ Certifications carousel with navigation
✓ Contact section with all details
✓ Fully responsive design
✓ Smooth scroll animations
✓ No external dependencies

HOW TO USE:
-----------
1. Add your profile image to the assets/ folder and name it "profile.jpg"
2. Open index.html in any modern web browser
3. That's it! No build process or server required.

DEPLOYMENT:
-----------
You can deploy this to any static hosting service:
- GitHub Pages
- Netlify
- Vercel
- CloudFlare Pages
- Or any web server

Just upload all the files and folders maintaining the same structure.

CUSTOMIZATION:
--------------
- Colors and theme: Edit CSS variables in styles/main.css (lines 1-80)
- Content: Edit text directly in index.html
- Animations: Adjust timing in styles/main.css and scripts/main.js
- Dark mode: Automatically handled with localStorage persistence

BROWSER SUPPORT:
----------------
Works on all modern browsers:
- Chrome/Edge 90+
- Firefox 88+
- Safari 14+
- Opera 76+

NOTES:
------
- The profile image path is set to "assets/profile.jpg" - make sure to add it!
- All animations respect user's "prefers-reduced-motion" setting
- The website is fully accessible and SEO-optimized
- No jQuery, React, or other frameworks - pure vanilla JavaScript!

Enjoy your static portfolio website!
